📌 Langkah-langkah Pengembangan:

1. Siapkan Project di GitHub

  • Buat repository baru di GitHub.

  • Aktifkan GitHub Pages pada repository tersebut agar bisa diakses secara publik.

Anda bisa menggunakan salah satu library berikut untuk memudahkan pembacaan file Excel:

  • SheetJS (XLSX.js)

  • ExcelJS

Rekomendasi:

  • SheetJS lebih sederhana dan sangat cocok untuk kebutuhan ini.

4. Kode Dasar untuk Menggabungkan Sheet Excel

Berikut contoh implementasi sederhananya:

index.html

html
<!DOCTYPE html>
<html>
<head>
  <title>Excel Sheet Merger</title>
  <script src="lib/xlsx.full.min.js"></script>
  <link rel="stylesheet" href="style.css">
</head>
<body>
  <h2>Excel Sheet Merger</h2>
  <input type="file" id="upload" accept=".xlsx" />
  <button onclick="mergeSheets()">Gabungkan Sheet</button>
  <br><br>
  <a id="download-link" style="display:none;">Unduh Excel Gabungan</a>

  <script src="script.js"></script>
</body>
</html>

script.js

javascript
function mergeSheets() {
  const input = document.getElementById('upload');
  if (!input.files[0]) {
    alert("Pilih file Excel terlebih dahulu!");
    return;
  }

  const reader = new FileReader();
  reader.onload = function (e) {
    const data = new Uint8Array(e.target.result);
    const workbook = XLSX.read(data, {type: 'array'});

    let mergedData = [];
    workbook.SheetNames.forEach(sheetName => {
      const sheet = workbook.Sheets[sheetName];
      const sheetData = XLSX.utils.sheet_to_json(sheet);

      mergedData = mergedData.concat(sheetData);
    });

    // Proses duplikasi SEP dengan keterangan berbeda
    const result = {};
    mergedData.forEach(row => {
      const sep = row['nomer sep'];
      const keterangan = row['keterangan'];

      if (result[sep]) {
        if (!result[sep].includes(keterangan)) {
          result[sep] += `, ${keterangan}`;
        }
      } else {
        result[sep] = keterangan;
      }
    });

    const finalData = Object.keys(result).map(sep => ({
      'nomer sep': sep,
      'keterangan': result[sep]
    }));

    const newWorkbook = XLSX.utils.book_new();
    const newWorksheet = XLSX.utils.json_to_sheet(finalData);
    XLSX.utils.book_append_sheet(newWorkbook, newWorksheet, 'Hasil Gabungan');

    const wbout = XLSX.write(newWorkbook, {bookType: 'xlsx', type: 'array'});
    const blob = new Blob([wbout], {type: 'application/octet-stream'});

    const url = URL.createObjectURL(blob);
    const downloadLink = document.getElementById('download-link');
    downloadLink.href = url;
    downloadLink.download = 'hasil-gabungan.xlsx';
    downloadLink.style.display = 'inline';
    downloadLink.textContent = 'Klik untuk mengunduh file gabungan';
  };

  reader.readAsArrayBuffer(input.files[0]);
}

5. Deploy ke GitHub Pages

  • Commit file tersebut ke repository GitHub.

  • Aktifkan GitHub Pages melalui tab Settings di repository Anda:

    • Pilih sumber dari branch utama (biasanya main atau master).

    • Tunggu beberapa menit, lalu akses URL yang disediakan GitHub Pages.

🚀 Workflow Penggunaan:

  1. Buka halaman web Anda dari GitHub Pages.

  2. Pilih file Excel dengan beberapa sheet.

  3. Klik tombol Gabungkan Sheet.

  4. File hasil gabungan akan bisa Anda unduh secara langsung.

B. Upload File ke Repository

  1. Masuk ke repository GitHub Anda.

  2. Klik "Add file" > "Upload files".

  3. Upload ketiga file:

    • index.html

    • script.js

    • xlsx.full.min.js

  4. Klik "Commit changes".

Langkah 3: Aktifkan GitHub Pages

  1. Masuk ke Settings di repository Anda.

  2. Cari menu Pages (di sidebar kiri, biasanya bagian "Code and automation").

  3. Pada bagian Source, pilih branch:

    • Source: main (atau master, sesuai default repo Anda)

    • Folder: / (root)

  4. Klik Save.

  5. Setelah beberapa detik, akan muncul URL GitHub Pages. Contoh:

    arduino
    https://denysankara.github.io/excel-sheet-merger/

Langkah 4: Coba Web Converter Anda

  1. Buka URL GitHub Pages Anda.

  2. Upload file Excel dengan beberapa sheet.

  3. Klik Gabungkan Sheet.

  4. Download file hasil gabungan.
